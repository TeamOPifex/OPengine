========================================
== Requirements
========================================

* CMake v2.8

--- Linux (Ubuntu 12.04 LTS) ---

* OpenGL development headers + lib (GL 3.3 capable drivers)
* OpenAL development headers + lib
* GCC & G++

--- Windows ---

* OpenAL Redistributable (oalinst)
* OpenAL SDK (OpenAL11CoreSDK)
* Visual Studio 2010/2012

--- Android ---

* Android SDK
* Android NDK
* Ant
